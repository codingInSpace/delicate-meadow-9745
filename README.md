## Usage
```js
import rose from 'violet-sine-rose'  

const options = {
  width: window.innerWidth,
  height: 600,
  respondToWidth: true,
}

someDomElement.appendChild(rose(options))  
```

## Develop
```bash
yarn
```   

Run dev server
```bash
yarn start
# localhost:8080/webpack-dev-server/bundle
```   

## Build for production  
```bash
yarn build
```
