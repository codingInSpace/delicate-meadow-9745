## Usage
```js
import rose from 'violet-sine-rose'  
const width = window.innerWidth  
const height = 600  
someDomElement.appendChild(rose(width, height))  
```

## Develop
```bash
yarn
```   

Run dev server
```bash
yarn start
# localhost:8080/webpack-dev-server/bundle
```   

Build for production  
```bash
mkdir public  
yarn build
# load index.html
```
